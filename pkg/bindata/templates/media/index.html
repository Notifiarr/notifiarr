  • Tautulli Integration Explained
    This is entirely optional. The Notifiarr website also provides this same feature and allows you to map Plex users to custom names.

    Explained:

    Tautulli provides a feature that allows you to give your Plex users custom usernames. By integrating Notifiarr client with Tautulli, those custom usernames will be displayed in your chat server notifications. When not enabled, Plex usernames or email addresses are used instead.
    The Tautulli integration is used to provide a Plex username to custom name mapping in notifications.
  • Disable Plex or Tautulli by removing the URL or setting Timeout to Disabled. The client only supports one of each.
  • Disable service checks by settings Interval to Disabled.
